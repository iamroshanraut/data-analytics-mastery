## 📌 What is Programming?

Programming is the process of creating sets of instructions that tell a computer how to perform specific tasks.

These instructions, known as **code**, are written in programming languages that a computer understands and executes to carry out various operations such as:

- 🔍 Solving problems
- 📊 Analysing data
- 🖥️ Controlling devices

## 🔄 Python is Dynamically Typed

In Python, you **don't need to declare a variable's type** — just assign a value and Python figures it out automatically.

```python
x = 8                          # x is an integer
print(type(x))                 # <class 'int'>

x = "Python by Rishabh Mishra" # now x is a string
print(type(x))                 # <class 'str'>

x = 3.14                       # now x is a float
print(type(x))                 # <class 'float'>
```

> ⚠️ **Key insight:** The variable `x` doesn't change — only the value (and therefore the type) assigned to it changes. Python handles this automatically at runtime.
